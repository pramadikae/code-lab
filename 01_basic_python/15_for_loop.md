# For Loop

For loop digunakan untuk melakukan iterasi (perulangan) pada urutan data seperti list, string, atau range.

## Struktur Dasar
```python
for variabel in urutan:
    # kode yang diulang
```

## Contoh Penggunaan
```python
for i in range(5):
    print("Perulangan ke-", i)
```
