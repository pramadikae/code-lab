# Assignment Operators

Assignment operator digunakan untuk memberikan nilai ke variabel. Operator ini juga bisa digunakan untuk operasi sekaligus assignment.

| Operator | Contoh   | Sama dengan   |
|----------|----------|---------------|
| =        | a = 5    | a = 5         |
| +=       | a += 2   | a = a + 2     |
| -=       | a -= 2   | a = a - 2     |
| *=       | a *= 2   | a = a * 2     |
| /=       | a /= 2   | a = a / 2     |
| //=      | a //= 2  | a = a // 2    |
| %=       | a %= 2   | a = a % 2     |
| **=      | a **= 2  | a = a ** 2    |

## Contoh Penggunaan
```python
a = 10
a += 5
print(a)  # 15
a *= 2
print(a)  # 30
a -= 10
print(a)  # 20
```
