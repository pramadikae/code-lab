# Function

Fungsi adalah blok kode yang dapat digunakan berulang kali. Fungsi didefinisikan dengan kata kunci `def`.

## Struktur Dasar
```python
def nama_fungsi(parameter):
    # kode
    return hasil
```

## Contoh Penggunaan
```python
def sapa(nama):
    print(f"Halo, {nama}!")

sapa("Andi")
sapa("Budi")
```
