# String

String adalah tipe data di Python yang digunakan untuk menyimpan teks. String ditulis di antara tanda kutip tunggal ('...') atau ganda ("...").

## Contoh Penggunaan String
```python
teks1 = 'Halo'
teks2 = "Dunia"
print(teks1)
print(teks2)
```

## Operasi pada String
- Menggabungkan string: `+`
- Mengulang string: `*`

```python
nama = "Python"
print(nama + " mudah!")
print(nama * 3)
```
